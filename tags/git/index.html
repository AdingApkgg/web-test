Git 创建仓库

创建一个 git 仓库有如下几种方式：

  • git init：初始化一个 git 仓库
  • git clone：clone 一个 git 仓库

下面对这几种方式进行详细介绍：

1. git init

Git 使用 git init 命令来初始化一个 Git 仓库，执行完 git init 命令后，会生成一个.git 目录，该目录包含了资源数据，且只会在仓库的根目录生成。

如果用当前目录作为 Git 仓库，则只需要执行如下命令：

git init
